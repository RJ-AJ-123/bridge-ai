# BridgeAI

## Agent skills

### Issue tracker

Issues live as local markdown files under `.scratch/<feature-slug>/`. See `docs/agents/issue-tracker.md`.

### Triage labels

Five canonical triage state labels at their defaults; project-specific type labels (bug, feature, enhancement, docs, refactor, test, blocked) are tracked separately. See `docs/agents/triage-labels.md`.

### Domain docs

Single-context — `CONTEXT.md` and `docs/adr/` live at the repo root. See `docs/agents/domain.md`.
